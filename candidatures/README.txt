DOSSIER DES CANDIDATURES
========================

Ce dossier contient les CV et lettres de motivation des candidats.

Structure :
-----------
Chaque candidature est stockée dans un sous-dossier avec le format :
YYYY-MM-DD_HHMMSS_NomDuCandidat/
    ├── CV_NomDuCandidat.pdf
    └── Lettre_Motivation_NomDuCandidat.pdf

Le fichier candidatures_log.txt contient un historique de toutes les candidatures.

Sécurité :
----------
- Ce dossier est protégé par .htaccess
- L'accès direct via navigateur est bloqué
- Seuls les scripts PHP peuvent accéder aux fichiers

ATTENTION : Ne pas supprimer le fichier .htaccess !
